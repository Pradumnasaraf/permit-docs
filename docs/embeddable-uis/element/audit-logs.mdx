# Audit Logs Element

<span className="subtitle-custom">
	Track and trace user activity as a single secure, embeddable UI component.
</span>
<br />
<br />
The Audit Logs Element is an embeddable UI component that gives you insight into
who did what at any moment in time within the system. This element gives full control
over your applications, enforcing security and providing the user with a history
of events.
<br />
<br />
